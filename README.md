# 🚀 Talencia Global: PES Placement Intelligence Platform

An enterprise-grade, AI-powered digital intelligence system developed during the **Industry-Academia Engineering Initiative** at [Talencia Global](https://talenciaglobal.com).

This repository contains the source code for a production-scale platform that integrates complex data architectures, metadata-driven test automation, and an orchestrated multi-agent AI ecosystem.

## 🏗️ Program Architecture & Systems Developed

This project bridges the gap between academic theory and production engineering. I specifically architected and contributed to the following **3 Core Engineering Systems**:

### 1. Metadata-Driven Test Automation
*Directory: `/pytest_project`*
- **Description:** A highly scalable testing architecture designed to ensure absolute data integrity.
- **Tech Stack:** Python, Pytest.
- **Scale:** Automatically generates **2,000+ validation checks** across the entire parameter system, catching schema deviations and logic errors before they hit production.

### 2. Complete Agentic Ecosystem (AI Orchestration)
*Directory: `/LANGGRAPH`*
- **Description:** Transformed isolated AI tasks into a coordinated, multi-agent intelligence ecosystem. Built AI-driven research agents that collect, structure, and validate company intelligence data.
- **Tech Stack:** LangGraph, LangChain, Pydantic, Gemini, Groq, OpenRouter.
- **Highlights:** Utilized stateful ReAct loops for complex reasoning and enforced strict multi-layer verification using Pydantic schemas.

### 3. Enterprise Database & Cloud Infrastructure
*Directory: `/placement-intel-core-main`*
- **Description:** The backbone of the platform, scaling from parameter discovery to deployment.
- **Tech Stack:** React, Node.js, Supabase, PostgreSQL, Docker, FastAPI.
- **Highlights:** Distributed data across **100+ normalized relational tables** and packaged the AI/Backend systems into containerized services for cloud-native deployment.

## ⚙️ Key Learnings
- Transitioning from simple scripts to **enterprise-grade system design**.
- Orchestrating multi-model LLM architectures using **LangGraph**.
- Building reliable CI/CD pipelines and writing robust automated tests at scale.
