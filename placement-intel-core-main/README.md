# Placement Intelligence Platform

A modern, high-performance dashboard for college placement cells and students. This platform tracks company hiring timelines, interview experiences, and provides deep analytics on campus recruitment drives.

## Features
- 📊 **Dynamic Dashboard:** Real-time stats on active hiring, scheduled interviews, and results.
- 🏢 **Company Profiles:** Detailed intelligence on companies, their work culture, CTC, and hiring processes.
- 📅 **Placement Timeline Tracker:** Stay up-to-date with registration deadlines, aptitude tests, technical interviews, and HR rounds.
- 🔔 **Notification Center:** Smart alerts for upcoming deadlines and new announcements.
- 🔍 **Search & Filter:** Easily explore and categorize companies by sector, size, and hiring velocity.

## Tech Stack
- **Frontend:** React, Vite, TypeScript
- **Styling:** Tailwind CSS, shadcn/ui
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Data Fetching:** React Query

## Local Setup & Installation

To run this project locally on your machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Pallavi-18-rgb/placement-intelligence-platform.git
   ```

2. **Navigate into the project directory:**
   ```bash
   cd placement-intelligence-platform
   ```

3. **Install the dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Open the app in your browser:**
   Navigate to `http://localhost:8080` (or the port specified in your terminal) to view the application.
